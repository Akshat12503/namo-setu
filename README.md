# 🕉️ Namo Setu — Pilgrimage Tourism App & Website

A full-stack pilgrimage tourism platform with AI-powered itinerary planning,
temple discovery, puja booking, route planning, and devotee assistance.

## Tech Stack
- Frontend: Angular 17+
- Backend: ASP.NET Core 8 Web API
- Database: PostgreSQL
- AI: Google Gemini 2.5

## Modules
- Temple Discovery & Map
- AI Pilgrimage Itinerary Planner
- Travel & Hotel Booking
- Puja / Chadhava Services
- Agentic AI Devotee Assistant
- Group Yatra Planner
- Emergency Support
